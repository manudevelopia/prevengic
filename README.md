## Description
This application tries to support Chemical pollution contaminators calculation on Safety & Security report

## Application
This application has been developed using:
 - Java 
 - SpringBoot
 - Typescript
 - React
 - PostgreSQL
 
## Testing

 - JUnit
 - Jest
 - Enzyme
